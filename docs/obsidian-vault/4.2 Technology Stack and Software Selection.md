# 4.2 Technology Stack and Software Selection

## Phase 9 additions

- **ChromaDB** (`chromadb >= 0.4.0`) — persistent vector database for RAG pipeline. `PersistentClient` stores embeddings at `data/rag_chroma/`. See [[ChromaDB]].
- **sentence-transformers** (`>= 2.2.0`) — local embedding model (`all-MiniLM-L6-v2`) for ChromaDB documents. No additional API cost. See [[SentenceTransformers]].

Both are optional imports in `rag.py` — missing dependencies degrade gracefully.
